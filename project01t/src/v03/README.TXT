작업목표: 추상 클래스와 추상 메서드 적용
=> 메뉴를 처리해야 할 클래스의 공통 기능을 정의하고,
   반드시 그 클래스를 상속받도록 유도한다.
=> 사용자가 메뉴 번호를 입력했을 때 호출될 메서드의 이름을 지정한다. => service()
=> 개발자가 반드시 그 service() 메서드를 구현하도록 강제한다. => 추상 메서드 
=> 이점: 개발의 일관성을 확보할 수 있다. => 코드의 유지보수가 쉽다.

1. 각 메뉴를 처리하는 컨트롤러의 공통점을 뽑아 수퍼 클래스로 정의한다.
2. 그 클래스는 직접 사용할 일이 없기 때문에, 사용해서는 안되기 때문에
   추상 클래스로 선언한다.
3. 상속 연습: MenuControl, StorageMenuControl 클래스 작성.
4. 반드시 하위 클래스에서 작성해야 할 메서드가 있다면 추상 메서드로 선언하라! 
5. StudentControl과 ProjectControl 클래스가 StorageMenuControl을 상속받도록 한다.
